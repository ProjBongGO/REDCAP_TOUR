# 상세기능내역 3. 수익증권시세 상세기능명세

## a) 상세기능정보
- **상세기능 번호**: 3
- **상세기능 유형**: 조회 (목록)
- **상세기능명(국문)**: 수익증권시세
- **상세기능 설명**: KRX에 상장된 수익증권의 시세 정보를 제공
- **Call Back URL**: https://apis.data.go.kr/1160100/service/GetStockSecuritiesInfoService/getSecuritiesPriceInfo
- **최대 메시지 사이즈**: 4000 byte
- **평균 응답 시간**: 500 ms
- **초당 최대 트랙잭션**: 30 tps
